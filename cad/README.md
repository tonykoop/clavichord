# CAD Notes

`clavichord.scad` is an ILLUSTRATIVE-ONLY action-coupon scaffold — a topology
diagram of the tangent-key mechanism (a key lever lifting a brass tangent into
one string from below, the tangent defining the sounding length). Authority:
`pending_measurement` — see `visual-output-register.csv` (cad-coupon) and
`mcp-session-log.md`.

This packet RELEASES NO DIMENSIONS (design.md / README "Readiness Boundary").
Every parameter in the scad is an `estimate_pending_measurement` placeholder,
chosen only so the mechanism reads as a shape. Do NOT scale, measure, or lay out
tangent/key/string positions from this model. Dimensioned CAD is authorized only
after the `validation.csv` measurement gates close (see `drawing-brief.md`).
