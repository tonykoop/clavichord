# Clavichord Drawing Brief

This is a V5 L2 concept packet that releases NO dimensions. The drawing brief
therefore describes the FIRST drawings to make once measurement gates close —
it does not authorize dimensioned CAD now. See `cad/clavichord.scad` for the
illustrative-only action-coupon topology.

## First drawings (produce only after `validation.csv` gates close)

- One-key action coupon: key lever, balance/pivot, tangent, and one string
  course — the mechanism the coupon proves (VAL-001..VAL-003).
- Tangent contact detail: face geometry, adjustment, and string contact line —
  released only after bench tests fix the tangent spec (VAL-001).
- String-course / layout diagram: fretted-listed vs unfretted, with the
  note-conflict map — released only after VAL-005.
- Damping detail: non-speaking-segment damping placement (VAL-004).
- Case / bridge / hitch / pin field load-path layout — released only after the
  string-load and pinch-point review (VAL-007).

## Authority Notes

- Nothing here is a dimension. The action coupon in `cad/clavichord.scad` uses
  illustrative placeholders only; do NOT scale, measure, or lay out tangent
  positions from it.
- Each future drawing must cite a measured or reviewed source before it can
  carry authority (VAL-008 cut-list-promotion gate).
