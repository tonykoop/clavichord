# Image Generation Prompt

Concept visualization only; not fabrication authority.

## Hero Prompt

Photorealistic 3/4 product photograph of a finished small clavichord on a clean neutral studio background, a low rectangular wooden case with the lid open to reveal parallel strings, bridges, hitch pins, a compact keyboard, and visible brass tangents rising from the backs of the keys toward the string courses. Show the instrument as an expressive tangent-keyed string mechanism rather than a piano: slim natural-wood keys, dark sharps, warm oiled hardwood case, brass hardware, felt or cloth damping strips near non-speaking string segments, precise but handmade joinery, realistic bench scale cues, soft directional studio lighting, about a 50mm lens, high detail, photorealistic.

## In-Context Prompt

Photorealistic image of the clavichord staged on a small workshop table beside calipers, a one-key tangent test coupon, spare brass tangent blanks, and a tuning wrench, with a player gently pressing one key while the tangent remains in contact with the string. Natural window light, quiet luthier-shop setting, visible strings and damping, the scene emphasizing touch, after-pressure, and serviceable adjustment without showing any fake logos or measurement labels.

## Detail / Macro Prompt

Macro photorealistic close-up of a brass tangent just contacting a paired string course from below, the key lever partly depressed, non-speaking string segments muted by a narrow damping strip, polished tangent face, fine string windings, shallow depth of field, soft raking light revealing the contact point and the tiny gap around adjacent strings.

## Negative / Avoid

No text, no watermarks, no brand marks, no piano hammers, no impossible action geometry, no extra keys or strings that collide, anatomically plausible hands if shown.
